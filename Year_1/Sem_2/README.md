## 🧠 Semester Reflection & Gained Knowledge

### 1. Key Insights Gained
This semester shifted my focus toward system optimization, structured user experiences, and rigorous data logic. Transitioning from basic syntax to *Programming Technique II* and *Data Analysis* significantly deepened my ability to parse and process datasets systematically. Simultaneously, balancing *Computer Architecture* with *System Analysis & Design* provided a comprehensive perspective on both low-level hardware constraints and high-level enterprise system planning. Finally, studying *Human Computer Interaction* reinforced my problem-solving skills, proving that software efficiency must always go hand-in-hand with intuitive UI/UX structures to deliver true real-world value.

### 2. Actionable Suggestions for Improvement
Reflecting on my learning workflow, integrating automated unit testing or automated dataset assertions during the early programming phases would have helped catch structural code bugs much faster. Moving forward, I plan to combine these system analysis paradigms with modern collaborative platforms to ensure my project development workflows closely match industry-standard agile processes.
