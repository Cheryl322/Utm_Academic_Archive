## 🧠 Semester Reflection & Gained Knowledge

### 1. Key Insights Gained
This semester marked a massive turning point in my technical journey, shifting my focus toward advanced software engineering design and data intelligence frameworks. Transitioning from standard procedural programming to *Object Oriented Programming* fundamentally elevated my ability to design modular, highly reusable, and enterprise-ready software structures. Simultaneously, balancing *Data Analytics Programming* and *Data Mining* allowed me to build core **problem-solving skills** around data manipulation—learning how to clean, process, and model raw datasets to discover actionable real-world insights. Finally, studying *Operating System* rounded out my engineering baseline by providing deep insights into memory allocation and resource concurrency.

### 2. Actionable Suggestions for Improvement
Reflecting on the heavy programming logic encountered this semester, I realized that data processing pipelines and complex OOP systems can easily become difficult to debug without clean environment setup strategies. Moving forward, incorporating localized package isolation management and writing early assertion tests for data inputs would drastically speed up my personal project development velocity.
