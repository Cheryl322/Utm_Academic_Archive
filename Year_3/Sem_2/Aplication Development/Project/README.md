## 📌 Project Overview
**GeniusAqil-OS** is an integrated operational management system developed specifically for **Tadika Aqil Miqail (Genius Aulad Skudai)**. Built with a human-centric approach, this application streamlines daily administrative tasks, tracks student enrollment, manages teacher assignments, and ensures seamless data synchronization across the entire institution.

---

## 🛠️ Tech Stack & Architecture

This platform utilizes a modern, cross-platform technical stack engineered for high performance, reactive data binding, and relational integrity:

*   **Frontend Framework:** `Flutter` (Dart) — Delivering a responsive, cross-platform UI/UX layout ensuring smooth navigation for both desktop and mobile views.
*   **Backend & Database:** `Supabase` — Leveraging a powerful cloud-hosted PostgreSQL layer for real-time data streaming, relational transaction management, and secure user authentication.
*   **State Management & Architecture:** Layered Model-View-Controller pattern ensuring a clean separation of concerns, optimized query operations, and easy system maintenance.
